Reels videos (homepage)

Folder name: Videos (capital V) — dapat tumugma sa URL /Videos/... (importante sa Linux/Vercel).

Ilagay dito:
  video1.mp4
  video2.mp4
  video3.mp4

Optional: .webm versions (same base name) — unang susubukin ng browser ang .mp4.

Preview:
  - Ang preview sa carousel ay ang mismong unang frame ng video (walang shared static image).
  - Optional lang ang custom poster: sa REEL_VIDEOS, `poster: '/Videos/video1-poster.jpg'` kung gusto mong JPG imbes sa video frame.

Kung walang video file, makikita ang mensahe na hindi mahanap ang file — siguraduhing naka-copy ang .mp4 sa folder na ito.
